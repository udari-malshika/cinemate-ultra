CineMate Elite | Premium Cinema & Series ---------------------
A high-fidelity, luxury-themed streaming interface designed for an elite cinematic experience. This project showcases a sophisticated 
UI with glassmorphism, gold accents, and an integrated AI concierge system, all contained within a single optimized HTML5 file.

---Premium Features----
1.Elite UI/UX: Built with a "Premium Gold" aesthetic featuring custom glassmorphism (backdrop-blur), sophisticated typography
(Plus Jakarta Sans & Playfair Display), and cinematic hover effects.

2.AI Concierge (Gemini Integration): A built-in "Cinemax AI Concierge" modal allowing users to describe moods or vibes to find the 
perfect cinematic masterpiece using the Gemini 2.5 Flash API.

3.Single Page Navigation: A custom JavaScript router manages different views:

Home: Featuring a cinematic Hero Billboard and curated "Studio Highlights."
The Film Archive: A dedicated catalog for high-fidelity movies.

Series Collection: A gallery for binge-worthy narrative television.

4.Private Vault (Watchlist): A functional bookmarking system that persists user selections using localStorage.

5.Interactive Details Modal: Immersive content overlays providing director information, elite ratings, and high-definition previews.

6.Responsive Architecture: Fully optimized for all devices with a mobile-first approach using Tailwind CSS.
